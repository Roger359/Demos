# Conversion Formula

###  From pounds to: 
#### Kilograms = X / 2.2046
#### grams = X / 0.0022046
#### ounces = X * 16

### From Kilogram to:
#### Pounds = X * 2.2046
#### grams =  X * 1000
#### ounces = X * 35.274

### From Gram to:
#### kilograms = X / 1000
#### pounds = X * 0.0022046
#### ounces = X * 0.035274

### From Ounces to:
#### kilograms = X / 35.274
#### grams = X / 0.035274
#### pounds = X * 0.0625
